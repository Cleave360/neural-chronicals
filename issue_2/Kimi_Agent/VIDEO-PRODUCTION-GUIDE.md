## 💡 PRO TIPS

1. **Keep text on screen long enough to read** - Minimum 3-4 seconds per text block
2. **Use motion to maintain attention** - Slow zooms, pans on static images
3. **Match text to speech rhythm** - If using voiceover later
4. **Preview on mobile** - 70% of YouTube views are mobile
5. **Hook in first 5 seconds** - Start with the most compelling visual
